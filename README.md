
## NoteNow: A note taking app
![logo](https://github.com/sumitk-singh/Note-Now/blob/dev/src/assets/images/logo.png)
***

### Features

- Add a new note
- Edit & Delete Note
- Filter & Sort by priority, date 
- Change Card Color
- Add Tags to notes
- Trash & Archive
- Archive Notes
- Delete Notes
- Authentication
- Sign-up
- Login
- Logout
- Dark Mode
